--------------------------------------------------------------------------------
  5. CODE STANDARDS
--------------------------------------------------------------------------------

Components:
  - Angular 20 standalone components only (imports: [...] in @Component)
  - Use @if / @for / @switch template syntax (not *ngIf / *ngFor)
  - Use signal() and computed() for reactive state
  - Load data in ngOnInit with async/await + try/catch/finally
  - Always set loading = signal(true), error = signal<string|null>(null)
  - Show a loading state while fetching and an error state if it fails

Services:
  - Inject SupabaseService and AuthService via constructor
  - Call this.db.methodName() — never import the Supabase client directly

Styles:
  - Component-specific styles go in the component's .scss file
  - Global/shared styles only go in src/styles.css (do not add to it)
  - Use BEM naming: .block, .block__element, .block--modifier
  - Use Tailwind utility classes for layout, spacing, flex, grid
  - Use SCSS for hover transitions, animations, dynamic state, gradients
  - Never hardcode colours — always use CSS custom properties

--------------------------------------------------------------------------------
  6. SUPABASE SERVICE — AVAILABLE METHODS
--------------------------------------------------------------------------------

Import: import { SupabaseService } from '../../core/supabase.service';

  db.getLeaderboard()                          → Promise<LeaderboardEntry[]>
  db.getRoundScores()                          → Promise<RoundScore[]>
  db.getSchedule()                             → Promise<Round[]>
  db.getNextRound()                            → Promise<Round | null>
  db.getUsers()                                → Promise<User[]>
  db.addUser(name: string)                     → Promise<void>
  db.setUserActive(id: string, active: bool)   → Promise<void>
  db.getBetResults(scheduleId?: string)        → Promise<BetResult[]>
  db.upsertBetResult({ user_id, schedule_id,
    gross, apply_multiplier })                 → Promise<void>
  db.deleteBetResult(userId, scheduleId)       → Promise<void>

For auth operations use AuthService:
  auth.isAuthenticated                         → boolean (getter)
  auth.session()                               → Session | null (signal)
  auth.signIn(email, password)                 → Promise<AuthResponse>
  auth.signOut()                               → Promise<void>
